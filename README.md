# CS495_ChallengeProblem2
Web Based project built in python off of the concept of CS495_ChallengeProblem1 and CS495_PythonWeb

This project requires the packages: flask, os, pandas, and re.
The file to run is located in the first level of the src folder and is titled "main.py".

This project can be run from an IDE or command line, just ensure if you are using an IDE that you run main.py and not the project.
